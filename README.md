# 💜 Shelfies Book Club

> Um clube do livro para quem gosta de se perder em histórias intensas, misteriosas e cheias de atmosfera — da melancolia de mansões assombradas às paixões que consomem tudo pelo caminho.  
> A lista é atualizada mensalmente com o título escolhido pelo grupo.
